# MyLibHub

MyLibHub adalah situs web yang dirancang sebagai repositori dan sistem manajemen koleksi buku digital.

Aplikasi ini dikembangkan menggunakan bahasa pemrograman PHP dengan memanfaatkan framework Laravel.

![MIT License](https://img.shields.io/badge/Version-v1.0.0-blue)
## Fitur Utama
MyLibHub berfungsi sebagai repositori dan manajemen koleksi buku digital, berfokus pada operasi CRUD (Create, Read, Update, Delete):

- Create: Menambahkan buke ke dalam repositori.
- Read  : Menampilkan koleksi buku di repositori.
- Update: memodifikasi atau memperbarui detail dari buku yang sudah ada.
- Delete: Menghapus buku yang tidak lagi dibutuhkan dari repositori secara permanen.
